# Interactive Classroom API

> [Русская версия](docs/README-RU.md)

Backend API for the Interactive Classroom Platform - a web application for conducting interactive classes in university auditoriums.

## Overview

The API provides server-side logic for managing interactive classroom sessions. It enables teachers to create sessions and allows students to connect via QR code and participate in various interactive modes (Q&A, quizzes, chat, reactions).

## Key Features

- Session management (create, configure, end)
- Q&A mode with moderation
- Quizzes with multiple choice answers
- General chat for participants
- Reaction system (emoji feedback)
- Activity rating for participants
- WebSocket for real-time communication
- Telegram bot integration
- Support for anonymous and identified modes
