# Документация API

> [English version](../README.md)

Backend API для платформы интерактивных занятий Interactive Classroom Platform.

## Описание

Backend API обеспечивает серверную логику платформы для проведения интерактивных занятий в университете. Позволяет преподавателям создавать сессии, а студентам подключаться через QR-код и участвовать в различных интерактивных режимах.

## Основные функции

- Управление сессиями (создание, настройка, завершение)
- Режим вопросов и ответов (Q&A) с модерацией
- Викторины с вариантами ответов
- Общий чат для участников
- Система реакций (эмодзи-отклики)
- Рейтинг активности участников
- WebSocket для обмена данными в реальном времени
- Интеграция с Telegram-ботом
- Поддержка анонимного и идентифицированного режимов

## Технологии

- REST API
- WebSocket для real-time коммуникации
- База данных для хранения сессий, вопросов, результатов
- Интеграция с внешними сервисами (перевод, SSO)
